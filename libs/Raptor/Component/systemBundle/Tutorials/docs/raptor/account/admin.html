Por último te enseñaré algo del manejo del panel !!

Cuando termines tu proyecto debes proteger el panel de control
al que solo tendrá acceso un usuario admin especial ya que aqui están,
las funciones que utilizas en el desarrollo y la configuración del sistema.

Esto puedes hacerlo facilmente, solo debes abrir el menú de usuario y pulsar
en New Amin Credentials, estableces tu usuario y tu contraseña y listo

Nota
El proceso de autenticación para el acceso al panel de control no está ni estará
integrado al security manager de Raptor ya que el security manager es usado por módulos
de seguridad personalizados o el propio Syntarsus que viene con Raptor. Ten en cuenta
que esto implica autenticarse nuevamente para el panel de control si algún módulo de
seguridad implementa el security manager.
