Aquí vamos!!
La configuración de Raptor es el primer paso para su utilización.
Como puedes ver solo se establece la configuración para la conexión
con tu base de datos y algunos otros parámetros.
(Si el nombre de la base de datos no existe no importa, Raptor la creará por tí)
(Si no conoces el puerto de conexión por defecto déjalo en blanco, se conectará de todas formas)

El parámetro secret es importante que lo generes, con el se crearán tokens de seguridad
para la protección contra ataques y se utiliza además como llave para el el algoritmo
de encryptación.

El parámetro de lenguage establece el idioma por defecto en Raptor, ten en cuenta que
esto no tiene efecto en el panel de control de Raptor, solo se utiliza en el contexto
idiomático para tu aplicación

Por lo demás todo está listo !!
